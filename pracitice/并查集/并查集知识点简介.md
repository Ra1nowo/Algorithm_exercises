# 并查集

定义:并查集是一种树型的数据结构，用于处理一些不相交集合的合并及查询问题(即所谓的并和查)
我们可以用并查集来判断一个森林中有几棵树、某个节点是否属于某棵树等

> 它支持一下3种操作:
> - _初始化(initial)_
> - _并(merge)_
> - _查(find)_


初始化并查集:
```
int p[N],cnt;
for (int i = 1; i <= n; ++i) p[i] = i;
```

并(merge)
```
int rx = find(x), ry = find(y);
if (rx != ry) p[rx] = ry;cnt--;
```

查(find)
```
// 压缩路径
if (x != p[x]) p[x] = find(p[x]);
return x;
```
